# PirateBayRuby

Install:

	gem install pirate_bay_ruby

Search:

	pirate_bay_ruby.rb "Ubuntu"

Download:

	pirate_bay_ruby.rb "Ubuntu" 1


Sorts all searches by highest seeder count

## Installation

Add this line to your application's Gemfile:

    gem 'pirate_bay_ruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pirate_bay_ruby
